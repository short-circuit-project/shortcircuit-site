⚡ Live AI Image Generation

Image & Logo Generation

Created the entire Short Circuit visual identity using AI — including Shorty, the robot mascot, and the project logo. Started in ChatGPT with image generation to develop the character concept, poses, and brand assets, then used Claude to build the brand system around them. A real-world case study in using the right AI tool for the right job.

What I learned

"Different AI tools have different strengths. ChatGPT crushed image generation while Claude was better for strategy and code. The best results came from knowing which tool to reach for — not from forcing one tool to do everything."

Process

How every project works

Same process every time. The AI helps at every step, but the curiosity and the decisions are mine.

01

Ask a question

Every project starts with something I don't know how to do. That's the point — the question IS the content.

02

Build with AI

I sit down with Claude and work through it. Prompt, iterate, break things, fix things. Record everything.

03

Document it

The whole process goes into the newsletter and on video. What worked, what didn't, what I'd do differently.

04

Ship it

Put it live. Add it to the portfolio. Move on to the next question. Repeat.
